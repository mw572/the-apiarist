# The Apiarist

A browser-based beekeeping simulation. Run a UK apiary across the seasons —
weather and forage, swarms and pests, varroa and disease, queens, splits,
harvesting and selling honey, and growing your operation from one nucleus
colony into an apiary.

It is built to teach: every action opens a short guided explanation of what
it is, why it matters and when to do it, so you could genuinely take up
beekeeping after playing.

## Play

**[mw572.github.io/the-apiarist](https://mw572.github.io/the-apiarist/)**

Or clone this repo and open `index.html` — it runs straight from the file,
no server and no build step.

## Built with

Plain HTML, CSS and vanilla JavaScript. Zero dependencies.
